Submission Instruction

Before starting the uploading process, please prepare your data set and make sure the meta data is available with you which is associated with the raw data that your going to upload. The meta data should be in a proper format with required details. Please check below template for meta data file for field and museum type data. This repository will offer an open opportunity to researches worldwide to collect spectral data for their research purpose, therefor please make sure that you are providing is legitimate and it’s owned by you. Please don’t upload any data that is protected by copy rights or patents. In such case this repository won’t take any responsibility and won’t pay any penalties to any authorities.
